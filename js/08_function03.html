JavaScript Functions

    지역함수(local function) : 함수안에 정의된 내장 함수를 의미.
        - 포함된 함수 안의 블록 사이에서만 쓰일 수 있다.
        - 포함된 함수의 실행이 종료되면 지역 함수도 소멸된다.
        - 함수 내부에서 선언된 변수명과 변수값은 함수 외부에서는 사용할 수 없다.
        전역 함수(global function) : 여러 코드 블록에서 동시에 쓰일 수 있는 함수를 말한다.
        - 전역 변수 : 여러 코드 블록에서 동시에 쓰일 수 있는 변수를 말한다.
        함수 밖에서 선언된 변수의 경우 함수 내부의 코드 블록에서도 동시에 사용할 수 있다.
